This page tracks comments made by horizontal groups on the [loading data] specification. It only shows tracker issues the Group has not closed. (Issues may remain open in the Group tracker even if closed by the other WG in their repo.)

The sections and items within those sections are ordered by the last modified date of the items in the tracker repository (not the discussions). The mostly recently changed items appear highest. The link on the left of each line links directly to the issue where the discussion is taking place. The date indicates when the tracker item was modified, and the right-hand column links to the issue used to track the discussion in the github repo.
